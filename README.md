# Las Quebradas Hardware Store – Database

Relational database system designed for inventory, sales, and customer management for a hardware store.

## Technologies
- Microsoft SQL Server
- T-SQL

## Features
- Product inventory management
- Sales and purchase tracking
- Customer and supplier records
- Stored procedures and triggers

## Setup
1. Open SQL Server Management Studio
2. Run the schema script to create the database
3. Run seed scripts to populate initial data
